3-API STORE
WSHOP souhaite, via une API REST, avoir accès à une liste de magasins et pouvoir filtrer ou trier sur les données existantes.
Nous souhaitons également pouvoir ajouter, supprimer ou modifier le(s) magasin(s) de notre choix.
Un code commenté et une documentation sur le fonctionnement de l’API sont attendus.
Attention : Ne pas utiliser de framework SYMFONY ou LARAVEL.
